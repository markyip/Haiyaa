# Haiyaa! v1.0 Release Notes

Fuiyoh! The first official release is finally here! If you use a colander to drain your rice, you will probably love this app because you both make terrible life choices.

This project was built because your driving in Forza Horizon 6 brings so much sadness, it deserves to be properly roasted. Now, every single time you hit a wall, Uncle Roger's voice will be right there to make sure you feel the disappointment.

## 🔥 What's New (Features)

- **Instant Disappointment Engine:** Advanced telemetry listening technology. The moment you crash your car, the app immediately deploys Uncle Roger yelling "Haiyaa!" directly into your ears. Zero latency, maximum emotional damage.
- **The "Sadness Variety Pack" (Random Playback):** You can now put multiple `.wav` files into the `sounds` folder. The app will pick a random sound every time you crash. Every mistake a new flavor of disappointment!
- **Lazy Man's Launcher (`run.bat`):** Added an interactive Windows menu that even your niece or nephew can use. No need to type long commands, just double-click and start crying.
- **Built-in Audio Fixer (`convert_mp3_to_wav.py`):** Windows winsound API is very picky, like an Asian mother-in-law. It only likes `.wav` files. We included a script to convert sad `.mp3` files into glorious `.wav` files automatically.

## ⚠️ Bug Fixes & Adjustments

- **Fixed Local Loopback Issue:** For the people playing on the Xbox Game Pass / Microsoft Store version—Windows likes to block the game's data. A troubleshooting guide has been added to the README so you can exempt the game. No more silent crashes. Now you can be judged properly.

## 📦 How to Install

1.  Download the source code zip file (`Haiyaa.zip`) below.
2.  Unzip it (don't mess this up, it's easier than making fried rice).
3.  Read the `README.md` to configure your Forza Horizon 6 settings.
4.  Turn up your speakers, start driving, and prepare to face the audio judgment.

> "If you drive car into tree, you put leg down from chair. Don't put leg down from chair. Drive better."
